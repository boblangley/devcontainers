## OS Support

This Feature should work on recent versions of Debian/Ubuntu, RedHat Enterprise Linux, Fedora, Alma, and RockyLinux distributions with the apt, yum, dnf, or microdnf package manager installed.

`bash` is required to execute the `install.sh` script.

This Feature derives from Microsoft's Dev Container Go Feature under the MIT License. It intentionally preserves the upstream debugging privileges while omitting the unrelated Docker `--init` request so an image-provided init system can remain PID 1.
